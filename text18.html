APIs : Twitter

API Twitter, il y a 5-6 ans, permettait (en version gratuite) de 

  • récupérer tous les tweets correspondant à un ou plusieurs mots-clés, publiés dans la semaine précédente
  • Récupérer les informations associées à un compte (username, nombre de followers, description de profil, etc.)
  • récupérer tous les tweets publiés par un compte donné (urls des tweets, texte, date, image éventuelle, nombre de likes, nombre de retweets, etc.)
