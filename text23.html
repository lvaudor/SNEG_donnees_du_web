Inconvénients

  • Une API n’est pas toujours disponible
  • Nécessité (en général) d’avoir une clé d’authentification
  • L’API peut être payante
  • Documentation de qualité variable + hétérogénéité des APIs
  • Toutes infos visibles sur la page ne sont pas récupérables

Avantages

  • Conçu pour être utilisé par les « robots »
  • Les données renvoyées sont structurées
  • Il existe des ressources en ligne pour prendre en main l’API (+ API-clients)
